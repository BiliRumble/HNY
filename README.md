## 安装

此项目使用 [pnpm](https://pnpm.io/), 请确保你已经安装了最新的 pnpm

```bash
pnpm install
```

## 开发

```bash
pnpm dev
```

## 构建

```bash
pnpm build
```

## 部署

```bash
pnpm preview
```
